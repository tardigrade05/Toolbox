Procedure:

1. Install python in this site(https://www.python.org/downloads/) then install it.
2. Download ToolBox zip file in this site(https://github.com/tardigrade05/ToolBox), find "Code", then click. "Download ZIP"
3. Extract zip file and enter the password '13711'
4. Open cmd , run as administrators and change directory where the main.py file is located by doing this command ..   "cd <PATH_NAME>".
5. In cmd run this command "pip install -r requirements.txt", wait until it finish to install required dependencies.
6. To convert to executable file (.EXE) run this command , "auto-py-to-exe".
7. Tab will pop up,
    SCRIPT LOCATION = browse "main.py"
    CONSOLE WINDOW = Window based(Hide the console)
    ICON = browse "tool-box.ico"

    Then click "CONVERT .PY TO .EXE"

8. After finish "OPEN OUTPUT FOLDER"
9. Copy all files in "output" folder into the folder where your main.py is located
10. And test the app by clicking the main.exe, if no errors occurs its good to go, you can now create shortcut app to your deskstop.
